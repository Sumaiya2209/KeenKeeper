# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a responsive friendship management web application that helps users build and maintain meaningful relationships. Users can log interactions such as calls, texts, and video chats, and gain insights into their communication patterns through visual analytics.

---

## 🛠️ Technologies Used

- ⚛️ React.js  
- 🧭 React Router DOM  
- 🎨 Tailwind CSS  
- 📊 Recharts  
- 🔔 React Toastify  
- 🗂️ Context API  
- 📁 JSON Data Handling  

---

## ✨ Key Features

### 1. 👥 Friend Management
- Manage and view friends from a structured JSON dataset  
- Displays profile picture, status, tags, and contact details  
- Fully responsive design for all devices  

---

### 2. ⚡ Interaction Tracking
- Log Call, Text, and Video interactions  
- Automatically adds entries to the timeline  
- Includes date, type, and friend name with toast notifications  

---

### 3. 📊 Analytics & Timeline
- Pie chart visualization of interaction types  
- Filter timeline by Call, Text, or Video  
- Helps track and understand communication patterns
